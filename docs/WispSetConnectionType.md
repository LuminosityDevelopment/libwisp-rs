# Function: WispSetConnectionType

Sets the connection type (TCP/UDP) in the Wisp context.

- `ws_ctx` - mutable reference to the [`WispContext`](./WispContext.md).
- `conn` - a [`ConnectionType`](./ConnectionType.md) enum value.
